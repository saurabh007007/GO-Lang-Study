# Go Lang Study Notes

This repository contains personal notes, examples, exercises, and snippets for learning the Go programming language.

## Purpose

- Collect and organize concepts, patterns, and small projects while learning Go.
- Provide runnable examples and tests to reinforce learning.

## Repository layout

- notes/ — conceptual notes and explanations (topics, tips, FAQs)
- examples/ — small runnable programs demonstrating features
- exercises/ — practice tasks and solutions
- snippets/ — useful code snippets and helpers
- cheatsheet.md — quick reference

## Getting started (macOS)

1. Install Go:
   - brew install go
2. Verify:
   - go version
3. Run an example:
   - cd examples/<example-dir>
   - go run .
4. Run tests:
   - go test ./...

Use Go modules (go mod init) for new examples or projects.
